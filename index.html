Quick Sort

i Time: O(n log n)
Space: O(log n)
Stable: No
Fastest in practice for most datasets.
O(n log n)
Comparisons: 0 Swaps: 0 
quickSort(arr, low, high) {
  if (low < high) {
    p = partition(arr, low, high);
    quickSort(arr, low, p-1);
    quickSort(arr, p+1, high);
  }
}

Merge Sort

i Time: O(n log n)
Space: O(n)
Stable: Yes
Guaranteed speed even in worst-case.
O(n log n)
Comparisons: 0 Swaps: 0 
mergeSort(arr, l, r) {
  if (l < r) {
    m = l + (r-l)/2;
    mergeSort(arr, l, m);
    mergeSort(arr, m+1, r);
    merge(arr, l, m, r);
  }
}

    swap(arr[0], arr[i]);
    heapify(arr, i, 0);
  }
